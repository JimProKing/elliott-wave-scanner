# ngrok으로 외부 공개하는 방법 (가장 쉬움)

당신의 컴퓨터에서 스캐너를 돌리면서, 다른 사람도 인터넷으로 볼 수 있게 해주는 방법입니다.

## 준비물
- Python + Flask (이미 설치했다고 가정)
- ngrok (무료)

---

## 1. 한 번만 해보기 (수동)

### 터미널 1: 로컬 서버 실행
```bash
cd C:\Users\a\Documents\Trading\elliott-wave-scanner
python local_server.py
```

이 창은 **계속 켜두세요**.

### 터미널 2: ngrok 터널 열기
```bash
ngrok http 5000
```

ngrok이 실행되면 이런 화면이 나옵니다:

```
Forwarding                    https://abc123.ngrok-free.app -> http://localhost:5000
```

**https://abc123.ngrok-free.app** 이 주소가 바로 외부에서 접근할 수 있는 주소입니다.

이 주소를 복사해서 친구나 동료에게 공유하세요.

---

## 2. 더 편하게: 배치 파일로 한 번에 실행

`start_with_ngrok.bat` 파일을 **더블클릭**하면 됩니다.

(이미 만들어 놓았습니다.)

이 배치 파일은:
- ngrok을 새 창에서 자동 실행
- 로컬 서버를 자동 실행

ngrok 창에서 https 주소를 복사해서 공유하면 끝.

---

## 3. 사용법

1. 배치 파일 더블클릭 (또는 위 2개 터미널 실행)
2. ngrok 창에서 https 주소 복사
3. 그 주소로 접속 → 모바일에서도 보기 편한 반응형 한국어 UI (표는 가로 스크롤)
4. **지금 스캔 실행** 버튼 클릭 → 당신 컴퓨터에서 실제 스캔이 돌면서 결과 업데이트
5. 다른 사람도 같은 주소로 들어와서 최신 결과를 볼 수 있음

**주의**: 
- 당신 PC가 꺼지거나 배치 파일을 종료하면 접속이 안 됩니다.
- ngrok 무료 계정은 주소가 가끔 바뀔 수 있어요 (유료는 고정 도메인 가능).

---

## 자주 묻는 질문

**Q. ngrok이 없어요**
→ https://ngrok.com/download 에서 다운로드 → 압축 풀고 ngrok.exe를 이 폴더에 넣거나, 시스템 PATH에 등록하세요.

**Q. https 주소가 안 나와요**
→ ngrok을 처음 실행할 때 계정 인증이 필요할 수 있습니다.
  `ngrok config add-authtoken <토큰>` 명령어로 인증하세요. (ngrok 사이트에서 무료로 받을 수 있음)

**Q. 다른 사람도 '지금 스캔 실행' 버튼을 누를 수 있나요?**
→ 네. 버튼을 누르면 **당신 컴퓨터**에서 스캐너가 실행됩니다. 결과는 모두에게 실시간 반영됩니다.

**Q. 보안이 괜찮나요?**
→ ngrok 무료 주소는 누구나 알면 들어올 수 있습니다. 중요한 데이터는 없지만, 민감하다면 ngrok에 Basic Auth를 걸 수 있어요:
  `ngrok http 5000 --basic-auth="아이디:비밀번호"`
